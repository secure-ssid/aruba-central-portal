# Retrieves assigned devices

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/sitemaps/:site-id/network-devices-assigned?filter=floorId eq 'dd5ccb1f-9b65-42a8-b50e-faa7842bbcd8'`

### Query Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| filter | floorId eq 'dd5ccb1f-9b65-42a8-b50e-faa7842bbcd8' | OData 4.0 filter must be provided with at least `floorId`. Other supported fields are  `types` (`in` operator must be used ), `mac`, `serial`, `name`
 |

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |
