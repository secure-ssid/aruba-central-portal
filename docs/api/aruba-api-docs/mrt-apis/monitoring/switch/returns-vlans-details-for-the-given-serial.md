# Returns vlans details for the given serial.

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/switch/:serial/vlans?site-id=2345678`

### Query Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| site-id | 2345678 | ID of the site for which switch-related information is requested. |

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |
