# Gateway cluster vlan mismatch summary

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/clusters/:cluster-name/vlan-mismatch`

### Query Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### Object containing information about the good and poor vlans, either for the entire cluster or for individual gateways within the cluster.


**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json
- `X-RateLimit-Limit`: 
- `X-RateLimit-Remaining`: 
- `X-RateLimit-Reset`: 

**Response Body:**

```json
{
  "good": 4,
  "poor": 2,
  "id": "gw-cluster-1",
  "type": "network-monitoring/gateway-monitoring"
}
```
