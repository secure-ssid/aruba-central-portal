# Gateway cluster connectivity graph information

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/clusters/:cluster-name/connectivity-graph`

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### Object containing information about gateways and its peers in cluster


**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json
- `X-RateLimit-Limit`: 
- `X-RateLimit-Remaining`: 
- `X-RateLimit-Reset`: 

**Response Body:**

```json
{
  "nodes": [
    {
      "name": "9240-VPNC-7011",
      "role": "Leader",
      "health": "Good",
      "serialNumber": "CNP5KYH02L",
      "peers": [
        {
          "name": "BCX1540221",
          "vlanMismatch": "No",
```
