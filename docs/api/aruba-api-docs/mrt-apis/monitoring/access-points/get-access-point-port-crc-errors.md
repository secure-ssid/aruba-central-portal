# Get Access Point Port CRC Errors

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/aps/:serial-number/ports/:port-index/crc-trends`

### Query Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### The response object contains transmission trend data for CRC Errors and percentage for a port of an access point.


**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json
- `X-RateLimit-Limit`: 
- `X-RateLimit-Remaining`: 
- `X-RateLimit-Reset`: 

**Response Body:**

```json
{
  "metric": "crc",
  "graph": {
    "keys": [
      "crc",
      "crc_percentage"
    ],
    "samples": [
      {
        "data": [
          1,
          50
        ],
        "timestamp": "2022-07-30T20:00:00Z"
      }
    ]
  },
  "id": "AP00000001/ports/0",
  "type": "network-monitoring/access-point-monitoring"
}
```
