# Get a list of Access Point WLANs

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-monitoring/v1alpha1/aps/:serial-number/wlans`

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### List of WLANs of an access point, along with their details.

**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json
- `X-RateLimit-Limit`: 
- `X-RateLimit-Remaining`: 
- `X-RateLimit-Reset`: 

**Response Body:**

```json
{
  "items": [
    {
      "wlanName": "wlan-00006",
      "securityLevel": "Captive Portal",
      "id": "AP00000001",
      "type": "string"
    },
    {
      "wlanName": "wlan-00006",
      "securityLevel": "Captive Portal",
      "id": "AP00000001",
      "type": "string"
    }
  ],
  "count": 1,
  "total": 10
}
```
