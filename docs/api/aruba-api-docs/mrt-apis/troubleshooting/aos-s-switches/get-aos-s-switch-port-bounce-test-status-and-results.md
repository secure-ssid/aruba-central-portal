# Get AOS-S Switch Port Bounce test status and results

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-troubleshooting/v1alpha1/aos-s/:serial-number/portBounce/async-operations/:task-id`

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### OK. Returns the status and results of the AOS-S Switch Port Bounce test.

**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json

**Response Body:**

```json
{
  "endTime": "2025-05-21T04:42:45.273878693Z",
  "status": "COMPLETED",
  "startTime": "2025-05-21T04:42:44.345783226Z",
  "failReason": null,
  "output": {
    "results": [
      {
        "port": "5-7",
        "status": "Bounced"
      }
    ],
    "ports": [
      "5-7"
    ]
  },
  "progressPercent": 100
}
```
