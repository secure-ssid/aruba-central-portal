# Get CX Switch Aaa test status and results

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-troubleshooting/v1alpha1/cx/:serial-number/aaa/async-operations/:task-id`

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### OK. Returns the status and results of the CX Switch Aaa test.

**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json

**Response Body:**

```json
{
  "status": "COMPLETED",
  "progressPercent": 100,
  "startTime": "2025-05-21T17:07:24.895362597Z",
  "endTime": "2025-05-21T17:07:31.123316017Z",
  "output": {
    "authenticationStatus": "accept",
    "username": "005056bd4115",
    "authMethodType": "chap",
    "radiusServerIp": "10.1.1.10"
  },
  "failReason": null
}
```
