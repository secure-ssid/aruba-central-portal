# Get AP Nslookup test status and results

## Request

**Method:** `GET`

**URL:** `{{baseUrl}}/network-troubleshooting/v1alpha1/aps/:serial-number/nslookup/async-operations/:task-id`

### Headers

| Key | Value | Description |
|-----|-------|-------------|
| Accept | application/json |  |

### Response Examples

#### OK. Returns the status and results of the AP Nslookup test.

**Status:** 200 OK

**Response Headers:**

- `Content-Type`: application/json

**Response Body:**

```json
{
  "startTime": "2025-05-20T13:26:20.837887330Z",
  "endTime": "2025-05-20T13:26:21.719771353Z",
  "output": {
    "hostName": "www.google.com",
    "dnsServer": "8.8.8.8",
    "resolvedAddresses": [
      "142.251.46.164"
    ]
  },
  "failReason": null,
  "progressPercent": 100,
  "status": "COMPLETED"
}
```
---
